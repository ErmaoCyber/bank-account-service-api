# Bank Account Management Backend System

## Tech Stack
- Java 21
- Spring Boot
- Spring Data JPA
- PostgreSQL
- Swagger / OpenAPI

## How to Run
1. Ensure PostgreSQL is running
2. Update database credentials in application.properties
3. Run the application:
   mvn spring-boot:run
4. Open Swagger UI:
   http://localhost:8080/swagger-ui.html

## Notes
This project is a backend MVP demonstrating REST APIs, persistence, and transactional logic.
